Cornwall parkrun Event History

 

How to Search

Use the search box to find event history by first finishers.

Start typing in the box and after two characters the matching first finishers will appear in a drop down list. Select the first finisher to find.

To remove the search, click or tap on the X next to the selected option.

What is this table?

This table shows a summary of each of this event's parkruns.

The data

  • Event # - the run's event number; clicking on this, or the date, takes you to that particular week's results.
  • Date - the date of the parkrun.
  • Number of Finishers - Number of parkrunners who completed the course
  • Number of Volunteers - Number of volunteers who are recorded as having helped at the event.
  • First Finisher - Male and female first finishers with their times.

Results can be sorted and searched using the filters at the top of the table.
